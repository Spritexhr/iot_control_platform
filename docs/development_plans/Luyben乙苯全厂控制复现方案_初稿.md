## 一、最终范围

- 只做 L1 控制结构复现；
- 不做动态耦合或 Aspen 数值验证；
- 使用现有通用 Project；
- 不恢复旧 `eb_plant` 插件；
- 普通 P/PI/PID 使用 `ControlScheme`；
- P-only 使用 PI 且 `Ki=0`；
- 总苯比值、R/F、VPC 使用 `AutomationRule`；
- 不增加脚本冲突检查；
- 系统用于教学模拟，不面向真实生产。

## 二、实际 Project 归属

不新建 `EB-LUYBEN`。Luyben 乙苯流程配置在运行库现有 Project：

```text
EB 苯乙烯生产工厂
├── 01 制乙苯车间：原料、R1、DEB 回流、R2
└── 02 乙苯分离车间：C1、C2、循环苯和 EB 产品
```

03、04 车间不在本方案范围。

## 三、仪表编号

仪表采用“前缀 + 四位号码”，不使用短横线：

```text
PI0101 = PI + 01 车间 + 01 号压力指示仪表
```

最后两位在同一仪表前缀内按物料方向排列。同一物流的 FI/PI/TI 可以共用四位号码：

| 号码 | 物流 |
|---|---|
| `0101` | C1 循环苯返回 R1 |
| `0102` | 新鲜苯 |
| `0103` | 新鲜乙烯 |
| `0104` | R1 出口 |
| `0105` | DEB 回流至 R2 |
| `0106` | R2 出口至 C1 |
| `0107` | R1 蒸汽移热辅助系统 |

例如：

```text
FI0103 → FIC0103 → FV0103
TI0104 → TIC0104 → PIC0107
PI0201 → PIC0201 → PV0201
```

## 四、实施方式

### 普通回路

- `FIC0103`：新鲜乙烯流量；
- `LIC0101/LIC0102`：R1/R2 液位；
- `TIC0104`：R1 温度至蒸汽压力内环设定值；
- `PIC0201/PIC0202`：C1/C2 塔压；
- `LIC0201-LIC0204`：两塔塔釜和回流罐液位；
- `TIC0201/TIC0202`：C1 第 14 级、C2 第 20 级温度。

### 组合脚本

- `FRC0102 + VPC0101`：总苯/乙烯比和 DEB 回流修正；
- `FRC0201`：C1 回流/进料比；
- `FRC0202`：C2 回流/进料比。
